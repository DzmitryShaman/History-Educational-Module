Справка для работы с обучающе-тестирующей программой по дисциплине "История СССР"

  • При запуске программы открывается начальный экран , где нужно выбрать дальнейшее действия:
    Для прохождения теста следует нажать на кнопку "Начать"
    Преподователь может посмотреть успехи учащегося для этого необходимо нажать на кнопку "Для преподователя"
     
  • После нажатия на кнопку "Начать" следует ввести ваши данные


  • После выбора варианта откроется окно с вопросами. Для того чтобы ответить на вопрос следует нажать на вариант ответа и затем нажать на кнопку "Выбрать ответить".



  • После прохождения теста появляется окно с вашим результатом. В нем находится количество правильных ответов, неправильных и кнопка "Заново",которая отвечает за возврат к выбору теста

  • На главном экране в контекстном меню располложена кнопка лекции по нажатию которой открывается документ , в котором можно воспользоваться материалом по любой из тем, которые представлены в тесте


  • Открытый файл "Материалы"


  • При нажатии на кнопку "Для преподователя" откроется окно с вводом данных, а именно "логин и пароль". Для дальнейшего открытия окна следует ввести данные из индивидуального ключа, который бы выдан преподователю.


  • После ввода данных из ключа нажмите на кнопку "Войти" после чего начнётся проверка на правильность введения данных. Если данные введены верно начнётся загрузка (формирование) отчёта.
